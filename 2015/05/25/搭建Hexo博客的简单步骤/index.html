搭建Hexo博客的简单步骤

前期准备,注册github账号,并创建GitHub Pages
1:安装node.js
2:安装git
3:在你的计算机中创建一个工作根目录,比如我是将hexo目录创建在D盘的. D:\hexo
3:打开工作根目录,右键调出gitbash
4:输入npm install -g hexo
5:输入hexo init
6:输入npm install
7:检查ssh_key
ls -al ~/.ssh
如果没有请输入:ssh-keygen -t rsa -C “github注册邮箱”
输入passphrase:要记住,后面用
ssh-agent -s
ssh-add ~/.ssh/id_rsa,如果本行出错执行下面两行
eval ssh-agent -s
ssh-add
然后在用户目录中找到.ssh目录中的id_rsa.pub,打开并复制其内容添加到github的ssh key中
8:测试ssh_key是否成功
ssh -T git@github.com,如果输出hi yourname 那就正常了
9:添加用户信息
git config —global user.name “cnfeat”//用户名
git config —global user.email “cnfeat@gmail.com”//填写自己的邮箱
10:编辑本地根目录中的配置文件_config.yml,添加github仓库信息

deploy:
type: git
repository: git@github.com:you_name/your_name.github.io.git
branch: master

11:使用
hexo n添加新文章
hexo g生成文章
hexo d发布
hexo s打开本地服务器测试:localhost:4000

注意:如果发布出现deplorer not found git,请使用如下命令:
npm install hexo-deployer-git —save
